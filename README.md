# Career Stability Analyzer

Analyzes real-world employment data to evaluate college major stability.

## Features
- Live data fetching from FiveThirtyEight public GitHub repository
- Automated data cleaning and missing value handling
- Correlation analysis between salary and unemployment rates
- Custom Stability Score balancing income (60%) and job security (40%)
- Categorizes majors into Volatile, Stable, and Highly Secure

## How to Run
```
pip install pandas numpy
python main.py
```

## Tech Stack
- Pandas (data processing)
- NumPy (numerical analysis)
- FiveThirtyEight dataset (real employment statistics)
